# study_pio benchmark dataset

Generated by `benchmark/online_pipeline/benchmark.py build`.

- `instances.jsonl`: Study PIO inputs with `article_ids`, no embedded article text
- `gold.jsonl`: population, intervention/comparator, and outcomes gold text
- `article_index.jsonl`: article id to de-duplicated cleaned article file
- `articles/*.json`: cleaned article fixtures
- `source_manifest.json`: frozen raw source metadata
- `build_manifest.json`: reproducible build metadata
- `split_manifest.json`: split IDs and strategy
