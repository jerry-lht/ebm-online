Task definition:
Extract the intervention slot from the clinical question below.

I definition and scope:
- I means the treatment, intervention, exposure, diagnostic test, or management strategy being evaluated in the question.
- C means an alternative, control, placebo, usual care, no treatment, or other comparator option.
- Return concise slot values, not full sentence rewrites.

Boundary rules:
- Extract only the option or options that the question semantics present as the intervention being evaluated.
- Do not include comparator, population, outcome, or study design details in I.
- When the question has clearer semantic markers for intervention versus comparator, follow those markers.
- Dataset-alignment heuristic: when the question structure is "A compared with B", "A versus B", or "A or B" and there is no clearer semantic marker, prefer the first clinical option as I and leave later clinical options for C.
- Treat the dataset-alignment heuristic as a tie-breaker, not as a general PICO definition.

Rules:
- Use only information inferable from the question text.
- Do not invent evidence details that are not present in the question.
- Return only JSON in this exact format: `{{"interventions":["..."]}}`
- If no intervention information is stated, return `{{"interventions":[]}}`.
