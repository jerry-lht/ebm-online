Example 1
Outcome evidence is summarized as psychological interventions compared with inactive control across the review.
Correct output: {"comparisons": ["psychological interventions vs inactive control"]}
Incorrect output: {"comparisons": ["psychological interventions vs waiting list", "psychological interventions vs placebo", "psychological interventions vs usual care"]}
Why: keep the grouped review-level comparator family instead of expanding to atomic study-level controls.

Example 2
Outcome evidence is summarized as exercise compared with usual care or no treatment as one grouped comparator family.
Correct output: {"comparisons": ["exercise vs usual care or no treatment"]}
Incorrect output: {"comparisons": ["exercise vs usual care", "exercise vs no treatment"]}
Why: grouped vs atomic is the key boundary; preserve the grouped family when that is how the review synthesizes the comparison.

Example 3
Outcome evidence only supports a broad comparator family such as inactive control.
Correct output: {"comparisons": ["service system approaches vs inactive control"]}
Incorrect output: {"comparisons": ["service system approaches vs placebo"]}
Why: do not collapse broad vs narrow into an unsupported exact atomic control.

Example 4
The evidence snippet names outcome scales and subgroup labels but does not support a recoverable review-level comparison.
Correct output: {"comparisons": []}
Incorrect output: {"comparisons": ["intervention vs control"]}
Why: when no supported comparison can be recovered, return an empty array instead of hallucinating one.
