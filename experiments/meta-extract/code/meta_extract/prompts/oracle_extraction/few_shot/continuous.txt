Allowed direct field names: Experimental mean, Experimental SD, Experimental N, Control mean, Control SD, Control N.
Use only the row that matches the official_item result.
Work in this order:
1. Identify one target continuous result row or one target row block for the official_item.
2. Confirm that the mean, SD, and N values belong to the same outcome, same subgroup, and same timepoint.
3. Copy Experimental mean, Experimental SD, Experimental N, Control mean, Control SD, and Control N from that same row or row block only.
Worked example:
Table row block:
- Postintervention symptom score
- Experimental: 4.0 (2.7), n=152
- Control: 3.9 (2.4), n=152
Return: {"predicted_rows":[{"direct_extraction_fields":[{"field":"Experimental mean","value":"4.0"},{"field":"Experimental SD","value":"2.7"},{"field":"Experimental N","value":"152"},{"field":"Control mean","value":"3.9"},{"field":"Control SD","value":"2.4"},{"field":"Control N","value":"152"}]}]}
Worked example:
Table rows:
- Postintervention: Experimental 3.66 (0.43), Control 3.34 (0.32), n=28 and n=29
- Follow-up: Experimental 3.41 (0.40), Control 3.30 (0.35), n=28 and n=29
official_item timepoint = postintervention

Do not build a fake outcome row from cohort descriptions, study duration text, or arm-size-only headers.
Do not build one predicted row by taking mean and SD from one row and N from a different row, header, or timepoint unless the source clearly presents them as one row block for the same result.
If only N is visible, return no row.
If mean or SD for either arm is missing for the target row, return no row instead of guessing from nearby numbers.
