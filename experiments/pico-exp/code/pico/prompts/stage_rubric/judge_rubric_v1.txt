Evaluate one document's PICO extraction quality with a structured rubric.

Task constraints:
- Use only the provided abstract, gold texts, and predicted texts.
- Do not use external knowledge.
- Do not infer treatment effects.
- Use EBM-NLP framing where I means Intervention plus Comparator/Control mentions when present.
- Output JSON only.

Document id: {doc_id}

Abstract:
{abstract}

Gold texts:
- P: {gold_p}
- I: {gold_i}
- O: {gold_o}

Predicted texts:
- P: {pred_p}
- I: {pred_i}
- O: {pred_o}

Rubric values:
- completeness: complete | partial | missing
- accuracy: accurate | mixed | incorrect
- noise: low | medium | high
- granularity: appropriate | too_broad | too_narrow | mixed
- overall_verdict: strong | mixed | weak

For each label P, I, O and overall:
- Provide the five rubric fields.
- Provide reason as exactly one concise sentence focused on rubric evidence.

Return an object with this shape:
- top-level key `labels` with keys `P`, `I`, `O`
- each label object has keys: `completeness`, `accuracy`, `noise`, `granularity`, `overall_verdict`, `reason`
- top-level key `overall` with keys: `completeness`, `accuracy`, `noise`, `granularity`, `overall_verdict`, `reason`
