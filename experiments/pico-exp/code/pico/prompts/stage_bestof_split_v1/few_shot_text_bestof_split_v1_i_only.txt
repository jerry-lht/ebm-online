Task definition:
Extract intervention spans from the randomized controlled trial abstract below.

I definition and scope:
- I means the exact span(s) that name what participants received, what they were compared against, or the care/assignment condition of a trial arm.
- This includes explicitly mentioned interventions, treatments, drugs, procedures, devices, behavioral programs, exposure conditions, comparator groups, control groups, placebo, sham interventions, usual care, no treatment, standard therapy, and other clearly named trial-arm conditions.
- If both intervention arms and comparator/control arms are mentioned, extract both.
- If the abstract provides only one combined comparison phrase that cannot be naturally split, keep that exact combined phrase.

Rules:
- Extract only exact substrings from the abstract.
- Do not paraphrase or infer.
- Return only JSON in this exact format: `{{"interventions":["..."]}}`
- Example with one span: `{{"interventions":["placebo"]}}`
- Example with multiple spans: `{{"interventions":["acupuncture","usual care"]}}`
- If no explicit I span is stated, return `{{"interventions":[]}}`.
